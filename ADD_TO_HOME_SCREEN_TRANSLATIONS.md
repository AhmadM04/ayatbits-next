## Usage in Component

The `AddToHomeScreen` component automatically detects the user's selected language and displays the appropriate translations using the `useI18n()` hook:

```tsx
const { t } = useI18n();

// Example usage:
<h3>{t('addToHomeScreen.installPromptTitle')}</h3>
<p>{t('addToHomeScreen.installPromptText')}</p>
```

## Adding More Languages

To add translations for additional languages:

1. Open the respective language file (e.g., `messages/fr.json` for French)
2. Add the `addToHomeScreen` section before the `landing` section
3. Translate all the keys to the target language
4. Save and test

## Testing Translations

1. Change the language in the app's language selector
2. Wait for the "Add to Home Screen" prompt to appear (or click the floating button)
3. Verify all text is displayed in the selected language
4. Check the modal instructions are properly translated
